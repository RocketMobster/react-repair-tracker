
# RMA Repair Tracker

This project is a React + Vite + Tailwind CSS Progressive Web App (PWA).

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```

## Features
- React
- Vite
- Tailwind CSS
- PWA support (to be configured)
